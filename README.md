# Music Reviews #

In-progress RESTful API for logging and retrieving album reviews. The basic backend functionality is almost complete. Front end to come soon.
  
**Backend technologies/frameworks**: Java, Spring Boot, Spring Security, JUnit, Mockito.  
**Frontend technologies/frameworks**: HTML, CSS, TypeScript, React

Repository is public only for portfolio display.  

### Required before first release ###
* Unit and integration testing
* Front end
* Documentation

## What to expect ##
User-driven music review app similar to Letterboxd in which users can submit reviews for albums they've listened to with a score of 1-100 with optional comments. The API populates and updates the artist/album database as reviews are created.
